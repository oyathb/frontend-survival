# 개발 환경 세팅

## JavaScript

install Node.js

terminal에서 \[brew install node]&#x20;

version 확인 \[node -v] (v19.8.1)



install fnm

\[brew install fnm]



## TypeScript

install visual studio code

\[brew install visual-studio-code]



작업 폴더 만들기



npm 패키지 - Node.js의 패키지를 관리할 수 있는 도구

visual studio code terminal에서 \[npm init -y]
